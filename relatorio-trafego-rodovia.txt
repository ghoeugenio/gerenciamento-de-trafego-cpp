Para o redigir o código, foi utilizado o IDE NetBeans, em linguagem C++. Nas linhas iniciais foram declaradas as bibliotecas usadas e o valor máximo de carros dado pela questão armazenado numa variável, após isso foi utilizado a função fflush_stdin, para a limpeza de buffer do teclado. Feito isso, foi instanciado a base do código, a estrutura de dados para armazenar todas as informações sobre os carros cadastrados.
 No início da função principal, é declarada todas as variáveis utilizadas no programa, incluindo a estrutura de dados, após isso é usado um laço infinito, esse laço funciona para a utilização do break quando finalizado o uso das opções do menu, dentro desse laço, há o menu principal do programa, organizado e bem intuitivo, oferecendo opções do 0 ao 6 de acordo com as funções pedidas no enunciado. Caso selecionado a opção 1, é possível cadastrar um veículo, escrevendo seu nome, informar se é veículo de carga ou passeio, estado de origem, ano de fabricação, quantidade de passageiros e, caso seja um veículo de passeio, a faixa etária dos passageiros. Cada leitura de dados oferece instruções para os valores de entrada e não é possível o cadastro de usuários com o mesmo nome. No segundo menu pode-se alterar qualquer dado inserido pelo usuário sobre o veículo, escolhendo as opções de acordo com o que deseja alterar usando os números de 0 ao 5, não é possível registrar um nome em uso, quando for feita a mudança do tipo de veículo, de carga para passeio, é solicitado novamente a quantidade de passageiros e a suas faixas etárias, visto que para veículos de carga não é solicitado esse tipo de dado e para alterações do número de passageiro para carros de passeio, é necessário informar a faixa etária de todos que estão no veículo. Caso o carro informado não faça parte do banco de dados cadastrados, é informado ao usuário que o carro não foi encontrado. O terceiro menu permite ao usuário excluir carros cadastrados, inserindo seus nomes. A lógica usada foi a utilização de um looping para buscar o carro a excluir no vetor de carros cadastrados, caso o encontre esse carro é sobreposto pelo dado do carro da próxima posição do vetor, e é feito essa sobreposição sucessivamente até o final dos cadastros, após isso há um decremento da variável que controla a quantidade de carros, finalizando a exclusão. Caso o carro não seja encontrado, é informado ao usuário. No quarto menu, é mostrado ao usuário a porcentagem de automóveis, do tipo passeio e do tipo carga, a exibição dos dados é limitado a 2 casas decimais, e, por isso, é possível que porcentagens que forem dizimas periódicas não exiba o valor com alta precisão, já que não é feito o arredondamento e por isso ocorre a perda de dados muito pequenos. No quinto menu é possivel acessar os relatórios contendo dados sobre os cadastros, por meio de um outro menu contendo a listagem de todos os veículos cadastrados, listagem de veículos por estado de origem, quantidade de veículos por estado de origem, quantidade de veículos por ano de fabricação e quantidade de usuários que trafegam na rodovia por faixa etária, tudo baseado no enunciado da questão. Detalhe que na segunda opção do menu é pedido o estado em questão, para a listagem dos veículos pertencentes a esse estado.
